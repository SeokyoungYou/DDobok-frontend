# DDobok

주짓수 도복 큐레이팅을 위한 웹사이트인 DDobok(또복)의 프론트엔드 코드 저장소입니다.

- 백엔드 코드 저장소: https://github.com/eakyoungyu/DDobok-backend

## 프로젝트 살펴보기

### 1. 실행 방법

```
npm install
npm start
```
